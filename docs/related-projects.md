# Related Projects

## Purpose

List relevant ecosystems and tools that can inform FileGate design, interoperability, and benchmarking.

## Native File Dialog Extended

Cross-platform C/C++ native file dialog library. Useful as:

- comparison target,
- low-level behavior reference,
- minimal API baseline.

## Native File Dialog

Older C-based dialog abstraction. Useful for:

- historical API patterns,
- reduced-abstraction behavior comparison.

## tinyfiledialogs

Small C/C++ dialog library. Useful as:

- lightweight comparative target,
- fallback behavior reference.

## XDG Desktop Portal / FileChooser

Critical Linux portal system for sandboxed application resource access.

Relevance for FileGate:

- URI/path behavior differences,
- grant persistence,
- desktop-environment-dependent behavior.

## Web Platform Tests + File System Access API

Web conformance and behavior-testing ecosystem.

Potential lessons:

- test taxonomy design,
- conformance framing,
- structured cross-implementation comparison.

## MCP File Uploads and Agent-Mediated Selection

Adjacent area for future exploration where host-mediated file selection is exposed to AI agents.

Potential future relevance:

- capability-based access semantics,
- declarative file access contracts,
- cross-tool interoperability testing.
